# Visual Field - iOS磁场可视化应用

一个基于iOS平台的实时磁场数据采集、可视化和分析应用，使用ARKit进行增强现实磁场矢量显示。应用 90% 由 Copilot 编写，仍然在修复问题中。

- 基于ARKit的3D磁场矢量可视化
- 实时位置跟踪和磁场方向显示
- 沉浸式AR体验

## 系统要求

- **iOS版本**: iOS 14.0+
- **设备要求**: 支持ARKit的iPhone/iPad
